# SaaS Demo Frontend

Frontend for a multi-tenant SaaS application built with Angular.

The goal of this project is to explore reusable UI components, authentication flows, dynamic forms and business-oriented user interfaces.

## Features

### Authentication

* Auth0 login
* Session management
* Protected routes

### Authorization

* Role guards
* Module guards
* User impersonation

### User Experience

* Dark / Light mode
* Internationalization
* Unsaved changes protection

### Data Management

* Filtering
* Pagination
* Reusable table component

### Dynamic Features

* Custom fields
* Dynamic validation
* Tenant configuration
* User management

## Tech Stack

* Angular
* TypeScript
* RxJS
* SCSS
* Auth0
